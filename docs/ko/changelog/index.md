# 변경 이력

이 페이지는 pacs008 도구 키트와 이 문서 사이트의 주요 변경 사항을 추적합니다.

## 2026

### 2026년 3월

- ISO 20022 pacs 메시지에 관한 25개 이상의 Q&A 쌍이 포함된 종합 FAQ 페이지를 추가했습니다.
- 전체 pacs 패밀리에 대한 상세 기술 참고 자료가 포함된 PACS 메시지 설명 페이지를 추가했습니다.
- 60개 이상의 ISO 20022 및 결제 메시지 용어를 정의하는 용어집 페이지를 추가했습니다.
- 2026년 11월 CBPR+ 요구 사항을 다루는 구조화 주소 마감일 페이지를 추가했습니다.
- 콘텐츠 출처, 검토 프로세스 및 방법론을 설명하는 편집 정책 페이지를 추가했습니다.
- FAQPage, TechArticle, SoftwareApplication 및 HowTo JSON-LD 스키마 마크업을 추가했습니다.
- 22개 전체 언어에 대한 hreflang 주석으로 사이트맵을 확장했습니다.
- 모든 영어 페이지의 가독성을 개선했습니다 (Flesch Reading Ease 55-58, Flesch-Kincaid Grade 8-9).
- 접근성을 위해 모든 언어에서 374개의 빈 테이블 헤더 요소를 수정했습니다.
- 비교, 운영 매트릭스 및 버전 차이 테이블에 테이블 캡션을 추가했습니다.
- 히어로 이미지를 2.7 MB에서 32 KB로 최적화했습니다.
- Lighthouse 점수: 접근성 96-100, 모범 사례 100, SEO 100.

### 2026년 1월

- 8가지 ISO 20022 pacs 메시지 유형에 대한 문서가 포함된 pacs008.com을 출시했습니다.
- 22개 언어로 문서를 생성했습니다.
- hreflang 태그, 정규 URL 및 구조화 데이터 (WebSite, BreadcrumbList, WebPage)를 구현했습니다.
- REST API, CLI 및 Python 라이브러리 문서를 게시했습니다.
- IBAN 및 BIC 검증 문서를 추가했습니다.
- Docker 배포 가이드를 출시했습니다.

## 2025

### 2025년 12월

- PyPI에서 pacs008 Python 도구 키트 최초 릴리스.
- pacs.008.001.13 XML 생성 및 검증 지원.
- CSV, JSON, JSONL, SQLite 및 Parquet 데이터 입력.
- JSON Schema 및 XSD 검증 파이프라인.
- SWIFT 문자 규정 준수 확인.
- 자동화된 워크플로를 위한 FastAPI REST API.
