# ความเป็นส่วนตัว

คำชี้แจงความเป็นส่วนตัวนี้อธิบายวิธีที่ pacs008.com จัดการข้อมูลเมื่อคุณเยี่ยมชมเว็บไซต์

## ข้อมูลที่เราเก็บรวบรวม

pacs008.com ไม่เก็บรวบรวมข้อมูลส่วนบุคคล เว็บไซต์ไม่ใช้คุกกี้หรือเทคโนโลยีการติดตามเพื่อระบุตัวผู้เยี่ยมชมแต่ละราย

## การวิเคราะห์

บริการวิเคราะห์ของบุคคลที่สามอาจเก็บรวบรวมข้อมูลที่ไม่ระบุตัวตนและรวมกัน เช่น การดูหน้าเว็บและ URL ที่อ้างอิง ข้อมูลเหล่านี้ไม่สามารถใช้เพื่อระบุตัวผู้เยี่ยมชมแต่ละรายได้

## สิทธิ์ของคุณ

ขึ้นอยู่กับเขตอำนาจศาลของคุณ คุณอาจมีสิทธิ์เกี่ยวกับข้อมูลส่วนบุคคลของคุณ รวมถึงสิทธิ์ในการเข้าถึง แก้ไข หรือลบข้อมูล เนื่องจาก pacs008.com ไม่เก็บรวบรวมข้อมูลส่วนบุคคล สิทธิ์เหล่านี้จึงไม่น่าจะใช้ได้

## การเปลี่ยนแปลงคำชี้แจงนี้

คำชี้แจงความเป็นส่วนตัวนี้อาจได้รับการแก้ไขเป็นระยะ การเปลี่ยนแปลงใด ๆ จะถูกเผยแพร่บนหน้านี้

## ติดต่อ
