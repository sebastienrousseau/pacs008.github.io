# Pribasiya

Ang pahayag ng pribasiya na ito ay nagpapaliwanag kung paano pinangangasiwaan ng pacs008.com ang impormasyon sa iyong pagbisita.

## Impormasyong kinokolekta namin

Hindi nangongolekta ng personal na datos ang pacs008.com. Hindi gumagamit ng cookies o tracking technology ang site para tukuyin ang mga indibidwal na bisita.

## Analytics

Maaaring mangolekta ng anonymous na pinagsama-samang datos ang mga third-party na serbisyo ng analytics tulad ng mga page view at referring URL. Hindi magagamit ang datos na ito para tukuyin ang mga indibidwal na bisita.

## Ang iyong mga karapatan

Depende sa iyong hurisdiksyon, maaaring mayroon kang mga karapatan kaugnay ng personal na datos, kabilang ang karapatang ma-access, maitama, o maalis ang impormasyon. Dahil hindi nangongolekta ng personal na datos ang pacs008.com, maaaring hindi naaangkop ang mga karapatang ito.

## Mga pagbabago sa pahayag na ito

Ang pahayag ng pribasiya na ito ay maaaring baguhin paminsan-minsan. Ang anumang pagbabago ay ipo-publish sa pahinang ito.

## Makipag-ugnayan

Kung mayroon kang mga tanong tungkol sa pahayag ng pribasiya na ito, bisitahin ang [pahina ng pakikipag-ugnayan](/tl/contact/).
