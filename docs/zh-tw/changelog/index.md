# 變更日誌

本頁面追蹤pacs008工具套件和本文件網站的重大變更。

## 2026

### 2026年3月

- 新增包含25個以上關於ISO 20022 pacs訊息問答對的綜合FAQ頁面。
- 新增PACS訊息詳解頁面，包含整個pacs系列的詳細技術參考。
- 新增定義60多個ISO 20022和支付訊息術語的詞彙表頁面。
- 新增涵蓋2026年11月CBPR+要求的結構化地址截止日期頁面。
- 新增說明內容來源、審查流程和方法論的編輯政策頁面。
- 新增FAQPage、TechArticle、SoftwareApplication和HowTo JSON-LD結構描述標記。
- 擴展了網站地圖，為全部22種語言新增了hreflang註釋。
- 改善了所有英文頁面的可讀性（Flesch Reading Ease 55-58，Flesch-Kincaid Grade 8-9）。
- 修復了所有語言版本中374個空白表頭元素以提高無障礙性。
- 為比較表、營運矩陣表和版本差異表新增了表格標題。
- 將主圖從2.7 MB最佳化至32 KB。
- Lighthouse評分：無障礙性96-100，最佳實務100，SEO 100。

### 2026年1月

- 上線pacs008.com，提供8種ISO 20022 pacs訊息類型的文件。
- 產生22種語言的文件。
- 實作了hreflang標籤、標準URL和結構化資料（WebSite、BreadcrumbList、WebPage）。
- 發佈了REST API、CLI和Python函式庫文件。
- 新增了IBAN和BIC驗證文件。
- 發佈了Docker部署指南。

## 2025

### 2025年12月

- 在PyPI上首次發佈pacs008 Python工具套件。
- 支援pacs.008.001.13 XML產生和驗證。
- CSV、JSON、JSONL、SQLite和Parquet資料輸入。
- JSON Schema和XSD驗證管線。
- SWIFT字元合規性檢查。
- 用於自動化工作流程的FastAPI REST API。
