# 变更日志

本页面跟踪pacs008工具包和本文档网站的重大变更。

## 2026

### 2026年3月

- 添加了包含25个以上关于ISO 20022 pacs消息问答对的综合FAQ页面。
- 添加了PACS消息详解页面，包含整个pacs系列的详细技术参考。
- 添加了定义60多个ISO 20022和支付消息术语的术语表页面。
- 添加了涵盖2026年11月CBPR+要求的结构化地址截止日期页面。
- 添加了说明内容来源、审查流程和方法论的编辑政策页面。
- 添加了FAQPage、TechArticle、SoftwareApplication和HowTo JSON-LD模式标记。
- 扩展了站点地图，为全部22种语言添加了hreflang注释。
- 改善了所有英文页面的可读性（Flesch Reading Ease 55-58，Flesch-Kincaid Grade 8-9）。
- 修复了所有语言版本中374个空表头元素以提高无障碍性。
- 为比较表、运营矩阵表和版本差异表添加了表格标题。
- 将主图从2.7 MB优化至32 KB。
- Lighthouse评分：无障碍性96-100，最佳实践100，SEO 100。

### 2026年1月

- 上线pacs008.com，提供8种ISO 20022 pacs消息类型的文档。
- 生成22种语言的文档。
- 实现了hreflang标签、规范URL和结构化数据（WebSite、BreadcrumbList、WebPage）。
- 发布了REST API、CLI和Python库文档。
- 添加了IBAN和BIC验证文档。
- 发布了Docker部署指南。

## 2025

### 2025年12月

- 在PyPI上首次发布pacs008 Python工具包。
- 支持pacs.008.001.13 XML生成和验证。
- CSV、JSON、JSONL、SQLite和Parquet数据输入。
- JSON Schema和XSD验证管道。
- SWIFT字符合规性检查。
- 用于自动化工作流的FastAPI REST API。
